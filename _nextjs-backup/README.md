# Next.js Analyzer Page (Backup)

Это бэкап Next.js версии страницы analyzer, которая пока не используется.

Старый статический HTML сайт (public/) - это основной рабочий вариант.

## Что здесь:
- `analyzer.tsx` - React компонент страницы анализа
- `package.json.nextjs` - конфиг для Next.js проекта

## Чтобы запустить Next.js версию:
1. Переименовать `package.json.nextjs` в `package.json`
2. Запустить `npm install`
3. Запустить `npm run dev`

## ВАЖНО:
Пока используется СТАРЫЙ статический HTML из public/
Next.js версия - это будущая миграция.
